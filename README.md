## Description

Boilerplate for NestJs-NextJs application using [NestJS](https://github.com/nestjs/nest) framework and [Nest-Next](https://github.com/kyle-mccarthy/nest-next) module.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
